What are you working on now?

Since leaving school, I've had time to work on honing certain old skills and learning new ones. I'm always hunting for more knowledge and experience, and a few of the things I've been working on have helped me round out my skillset.

I've been working on an untitled FPS project in Unreal Engine 5. Because of the wealth of resources available right now, I've been able to progress rapidly. Coming from an engine like Unity makes me appreciate all the tools I have available even more.

Concept

Most of the work I've done in the short time since I've moved back has been in Blender, Substance Designer/Painter, and Unreal Engine 5. I have had a heavy focus on creating amazing looking hand painted textures that exentuate simple modeling technique. My hopes with this format is to be able to create assets that are actually usable in a game engine, and still look stunning!
